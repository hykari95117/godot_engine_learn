# 노드(Node)와 씬(Scene)
### 노드(Node)
    - 특정 기능을 가지고 있는 레고 블록
        - ex)
        - 스프라이트 노드: 이미지 표시
        - 카메라 노드: 화면에 보이는 장면 정의
    - 다양한 노드들을 조합해 게임 오브젝트나 환경을 구성한다.
    - 노드들의 조합은 부모-자식 관계를 표현할 수 있는 트리 구조로 관리된다.
### 씬(Scene)
    - 확장자 - tscn
    - 노드들의 조합으로 하나의 장면을 만든 것
    - 게임 내에서 특정 상황이나 환경을 나타내며 하나의 레벨, 캐릭터, UI 등 독립적인 요소가 될 수 있다.
    - Scene 또한 부모-자식 관계를 가질 수 있기에 재사용성이 높다.

# GDscript(python과 유사한 문법)
- 하나의 Node에는 하나의 script만 넣을 수 있다.
- 바꿔말하면 script는 Node에 속해야만 실행된다. scrip 파일만으로는 아무런 의미가 없다.
- Node도 Scene에 속해야만 의미를 갖는다.
    - 정리
        - 스크립트는 그 자체로 실행되지 않으며 노드에 붙어야 해당 노드와 함께 씬에 로드되며 코드가 활성화된다.
        - 그래야 스크립트의 함수 등이 정상 작동할 수 있다.
```gdscript
_ready()
    - 해당 노드가 씬 트리의 일부가 되고 모든 자식 노드가 준비되었을 때 딱 한 번 호출
    - 노드나 게임 요소를 초기화하기에 적합
```
